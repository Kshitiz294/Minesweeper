1. The field contains mines scattered all around randomly. Mark all the mines and open all other boxes to win the game.
  2. Left-click on a box to open it. If it contains a number, that number is the number of how many mines are surrounding it.
  3. Right-click on a box to mark it as mine.
  4. If you mark the mine, you can open "unopened" squares around it, opening more areas.
  5. If you open a box containg mine, the game ends!!!
